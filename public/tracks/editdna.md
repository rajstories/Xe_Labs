# EditDNA Track

EditDNA is an XE Labs Build Sprint 2026 challenge to build an AI video editing automation engine. The prototype should turn scripts, raw footage, supporting assets, and a reference style into a scene breakdown, b-roll plan, caption treatment, and structured editing timeline.

Teams should use copyright-safe assets and must not clone another creator's protected content, watermark, or brand identity.

Full challenge: https://xelabs.in/build-sprint/editdna
